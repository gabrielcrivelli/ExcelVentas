\# BI Retail Analytics



Plataforma de BI para analizar ventas de retail (hipermercados) con:

\- Carga de múltiples Excels mensuales (dos sucursales).

\- Consolidador y limpiador de datos.

\- KPIs por Departamento/Familia/SKU.

\- Clasificación ABC y análisis Pareto.

\- Proyecciones simples de stock.

\- App web en Streamlit con descarga de Excel según columnas elegidas.
